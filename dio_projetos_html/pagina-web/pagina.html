Inspetor de Elementos

(Voltar)

O "Inspetor de Elementos" do navegador é uma ferramenta poderosa para desenvolvedores da web e entusiastas que permite visualizar e modificar o código HTML, CSS e JavaScript de uma página da web em tempo real. Essa ferramenta é frequentemente usada para depuração, inspeção e otimização de páginas da web. Você pode acessá-la pressionando F12 ou clicando com o botão direito do mouse em uma página da web e selecionando a opção "Inspecionar" ou "Inspetor de Elementos". Com o Inspetor de Elementos, você pode:

  • Examinar o código-fonte HTML de uma página.

  • Visualizar e editar estilos CSS aplicados a elementos.

  • Depurar código JavaScript, definir pontos de interrupção e observar variáveis.

  • Testar e experimentar alterações temporárias na página.

  • Monitorar solicitações de rede, recursos carregados e tempos de carregamento.

Essa ferramenta é fundamental para desenvolvedores web, pois ajuda a identificar problemas, ajustar o layout, testar códigos e melhorar o desempenho de sites e aplicativos da web. Se você está começando a estudar programação e deseja atuar como desenvolvedor, familiarizar-se com o Inspetor de Elementos é uma habilidade valiosa, especialmente se estiver trabalhando com tecnologias web, como HTML, CSS e JavaScript.

Estrutura Básica HTML

(Voltar)

A estrutura do HTML é formada por um conjunto de elementos, ou seja, os hipertextos, que se conectam entre si formando a página. Os elementos HTML ou também chamados de tags HTML, são utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

Dessa forma, para que um documento seja interpretado pelo navegador, é necessário que o arquivo tenha a extensão .html e a partir disso, poderá ser exibido por qualquer navegador web.

O HTML (Hypertext Markup Language) é a linguagem padrão para criar páginas web. Sua estrutura básica consiste em:

  • !DOCTYPE html: Define a versão do HTML que está sendo usada (geralmente HTML5).

  • html: Elemento raiz que engloba todo o conteúdo da página.

  • head: Contém informações sobre o documento, como título e metadados.

  • title: Define o título da página.

  • body: Contém o conteúdo principal da página.

Atributos das Tags

(Voltar)

Atributos são informações adicionais fornecidas dentro das tags HTML que modificam ou fornecem contextos específicos para os elementos. Eles geralmente consistem em um nome e um valor, separados por um sinal de igual (=) e envolvidos por aspas duplas (ou simples). Alguns exemplos de atributos são:

Atributos

  • href: Endereço do link.

  • src: Endereço da imagem.

  • class: Nome da classe do elemento.

  • id: Nome do id do elemento.

  • type: Tipo do elemento.

  • name: Nome do elemento.

  • value: Valor do elemento.

  • placeholder: Texto de exemplo do elemento.

Textos

(Voltar)

Em HTML, os textos são representados principalmente por meio de elementos de texto, como parágrafos (p), títulos (h1 a h6), e spans (span). Além das informações já mencionadas, alguns elementos e conceitos adicionais são:

Elementos

  • span: Usado para aplicar estilos a partes específicas do texto.

  • br: Usado para quebrar linhas.

  • hr: Usado para criar uma linha horizontal.

  • blockquote: Usado para citações.

  • pre: Usado para texto pré-formatado.

  • code: Usado para texto de código.

  • em: Usado para texto enfatizado.

  • strong: Usado para texto forte.

  • small: Usado para texto pequeno.

  • sub: Usado para texto subscrito.

  • sup: Usado para texto sobrescrito.

    Links

    (Voltar)

    Links (a) são usados para criar links para outras páginas, arquivos, endereços de e-mail, seções da mesma página, entre outros.

    Links para seções dentro da mesma página são criados usando o atributo href com o valor sendo o identificador da seção desejada.

    Para abrir um link em uma nova aba ou janela, você pode usar o atributo target="_blank"

    Links de e-mail são criados utilizando o prefixo mailto: no atributo href.

    Os links podem envolver não apenas texto, mas também imagens. Isso é feito encapsulando a imagem dentro da tag (a).

    Para criar um link de download para um arquivo, você pode usar o atributo download na tag (a).
